LottoWin Application

 

Mit LottoWin lässt sich schnell ausrechnen ob man beim Lotto oder beim Joker etwas gewonnen hat.

Start

LottoWin erlaubt es mit dem Menupunk Update die neuesten Lottozahlen herunterzuladen. Dazu muss allerding eine Verbindung zum Internet bestehen. Falls dein Computer via Proxy auf das Internet zugreifft muss dies beim Aufruf angegeben werden (meinproxyserver und port durch die entsprechenden Angaben ersetzen).

mit JRE

ohne Proxy: jre -cp LottoWin.jar LottoWin 
mit  Proxy: jre -Dhttp.proxyHost=meinproxyserver -Dhttp.proxyPort=port  -cp LottoWin.jar LottoWin
wenn ein Java Development Kit (JDK) installiert ist
ohne Proxy: java -classpath %classpath%;LottoWin.jar LottoWin
mit  Proxy: java -Dhttp.proxyHost=meinproxyserver -Dhttp.proxyPort=port -classpath %classpath%;LottoWin.jar LottoWin

Am einfachsten ist es für den Start von LottoWin ein Batch-File zu erstellen.

Das Programm sollte eigentlich selbsterklärend sein (so hoffe ich wenigstens). Falls doch noch Fragen auftauchen, schreibt mir eine E-Mail.
