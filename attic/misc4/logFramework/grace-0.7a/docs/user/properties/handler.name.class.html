Name: log.handler.name.class
Description: This controls which class is loaded to handle a content type as specified in the log.handler.name.url property. Normally, the user should not change this.  The class used to handle the content of a url is controlled by the type of the url.
Type: String
Acceptable Values: The name of any valid sub class of grace.log.Handler.
Default Value: Depends on url given in the associatedlog.handler.name.url property.
Example Usage:
$ java -Dlog.handler.text-file.class="MyFileHandler" grace.log.Log Uses MyHandler instead of using grace.log.FileHandler.
