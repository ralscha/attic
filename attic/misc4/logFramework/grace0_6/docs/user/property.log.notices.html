Name: log.notices
Description: This switch turns on or off all logging of notices.  More technically, it enables or disables notice log events being delivered to all/any log handler.
Type: boolean
Acceptable Values:
true Enables notice log events to be delivered to all handlers
false Disables notice log events being delivered to handlers
 
Default Value: true
Example Usage:
$ java -Dlog.notices=true ... Enables notices log to be delivered to all handlers.
$ java -Dlog.notices=false ... Disables notices log events being delivered to handlers.
