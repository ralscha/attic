UNINSTALL of PROGRAM
--------------------
This information can be deleted by in one of
three ways:

1. Run the uninstall program:

     "destination directory\uninstall\Uninstall_Sample_Program"

   This will run the batch file which removes the files.

2. Click on the uninstall icon added to the
   start menu or desktop folder.

3. Go to the Add/Remove Programs dialog in the
   Windows Control Panel and remove the program.
