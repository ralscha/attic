Name: log
Description: This is the "master" switch that turns on or off all logging.  More technically, it enables or disables log events being delivered to all/any log handler.
Type: boolean
Acceptable Values:
true Enables all log events to be delivered to all handlers
false Disables all log events being delivered to handlers
 
Default Value: true
Example Usage:
$ java -Dlog=true ... Enables all log to be delivered to all handlers.
$ java -Dlog=false ... Disables all log events being delivered to handlers.
