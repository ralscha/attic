Name: log.errors
Description: This switch turns on or off all logging of errors.  More technically, it enables or disables error log events being delivered to all/any log handler.
Type: boolean
Acceptable Values:
true Enables error log events to be delivered to all handlers
false Disables error log events being delivered to handlers
 
Default Value: true
Example Usage:
$ java -Dlog.errors=true ... Enables error log to be delivered to all handlers.
$ java -Dlog.errors=false ... Disables error log events being delivered to handlers.
