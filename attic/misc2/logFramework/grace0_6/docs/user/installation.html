Installation

$Revision: 1.1 $
$Date: 1999/03/01 17:24:21 $

Installation is like any other Java package.  Unjar the downloaded file and set your classpath.  Don't unjar the classes.jar file unless you want a bunch of class files.

$ jar xvf grace-0.6.jar
 12091 Sun Mar 07 20:56:02 GMT+04:30 1999 META-INF/MANIFEST.MF
 67121 Sun Mar 07 20:38:54 GMT+04:30 1999 grace-0.6/classes.jar
     0 Sun Mar 07 20:29:36 GMT+04:30 1999 grace-0.6/docs/
     0 Sun Mar 07 19:49:18 GMT+04:30 1999 grace-0.6/docs/api/
...
$ export CLASSPATH=$CLASSPATH:$(pwd)/grace-0.6/classes.jar
$ java grace.log.Log        # Test
03/07/99 16:29:57(0.014):1: notice: system.main.Test-Thread.0: Log$Test.logSomeMessages:1187: Examples of trace()

03/07/99 16:29:57(0.066):2: trace: system.main.Test-Thread.0: Log$Test.logSomeMessages:1188

03/07/99 16:29:57(0.066):3: trace: system.main.Test-Thread.0: Log$Test.testFunction:1145

...

Thats it!  If you running a graphical shell ontop of DOS (otherwise known as Windows) then you're on your own.  I don't do DOS so I can't answer any questions if your running DOS.  No offense, but if you're running DOS, you really ought to think about upgrading to UNIX.  It's easy.  Just say "NO" to DOS.
