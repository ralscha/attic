Manipulate Wordlist Cases

This feature is designed to aid in manipulating wordlist cases. It gives you the options to change the first or last character of every word in a list to lower or upper case, or change each entire word to upper or lower case. To use this feature, simply run the "Manipulate Wordlist Cases" option from the main interface, choose an input wordlist, the type of manipulation you desire, and output file, and click the "Manipulate" button.
