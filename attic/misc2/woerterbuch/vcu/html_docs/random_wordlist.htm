Generate Random Wordlist

This function is made to create random wordlists with user-definable attributes. The wordlists this function generates can be used when attempting to crack password files. The options of this feature are quite similar to the Generate Single Password function. To use this feature, simply run the "Generate Random Wordlist" option from the main user interface, choose a type of wordlist to generate from the drop-down list, choose a word length, a number of words to generate, and an output file and click the "Generate" button.
