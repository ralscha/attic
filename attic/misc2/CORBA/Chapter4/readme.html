Count Readme

This sudirectory contains the VisiBroker for Java programs developed in Chapters 4 and 5. Prior to running the programs in this subdirectory, please review the installation and setup information in the overview readme for this CD-ROM.

Prerequisite Software

  • VisiBroker for Java 3.1
  • Symantec Visual Café 2.1
  • JavaSoft Java Development Kit (JDK) 1.1

Running the Programs

Follow the instructions for running the programs as described in the chapters for each of the sample programs.

  • CountClient.java
  • CountClientApplet.java
  • CountServer.java
