JDBC Bank Readme

This sudirectory contains the VisiBroker for Java and JDBC programs developed in Chapters 23-26. Prior to running the programs in this subdirectory, please review the installation and setup information in the overview readme for these sample programs.

Prerequisite Software

  • VisiBroker for Java 3.1
  • VisiBroker for Java Naming Service
  • Symantec Visual Café 2.1
  • SQL Server 6.5

Running the Programs

Follow the instructions for running the programs as described in the chapters for each of the sample programs.  You can also use the following batch files in the CorbaJavaBook.2e subdirectory to run the programs using the following steps:

start SQL Server via the Service Manager
prompt> RunCreateBank
prompt> start RunNaming
prompt> start RunGatekeeper
prompt> start RunCoordinator
then load MultiConsole.html from your browser.

Then run
prompt> start RunDebitCredit2TClient client-number database-server
            or
prompt> start RunDebitCreditServer database-server
prompt> start RunDebitCredit3TClient client-number

At this point you can control the execution of your clients using the MultiConsole.
